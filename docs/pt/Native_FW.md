# Firmware nativo

[Aqui está o firmware nativo para as impressoras AD5M, AD5MPro e AD5X](https://github.com/ghzserg/FF/releases/R)

## Como instalar o firmware nativo

1. Redefina a impressora para as configurações padrão
2. formate o flash USB para FAT/FAT16/FAT32
3. Coloque o arquivo na pasta raiz do USB Flash
4. Desligue a impressora
5. Insira a unidade flash USB na impressora
6. Ligue a impressora

## Instalação do firmware completo no AD5X

Esta não é a firmware original, mas uma firmware de recuperação (para "desbrickar"). Deve ser instalada apenas se a impressora não ligar de forma alguma.

A instalação leva muito tempo, até uma hora.

Download em um pendrive USB

- [AD5X-factory-1.0.9.tgz](https://github.com/ghzserg/FF/releases/download/R/AD5X-factory-1.0.9.tgz)

## Instalar o XY

Para aqueles que foram para a "casa" deslocada do telhado (estacionamento para o receptor de cocô). Extraia o conteúdo do arquivo para a raiz da unidade flash, insira a unidade flash e reinicie a impressora. A calibração será carregada. Primeiro, pressione Reset. Em seguida, defina a posição correta do cabeçote usando os botões de controle na tela. Fixe-a com o botão Set. Reinicie a impressora sem a unidade USB. Arquivo do suporte oficial do flashforge.

[Set.XY.Offset.zip](https://github.com/ghzserg/FF/releases/download/R/Set.XY.Offset.zip)
