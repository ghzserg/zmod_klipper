# Список макросов Z-Mod

Макрос - это небольшая программа на языке Klipper/Gcode. 

Он может вызываться:

- Из файла GCODE
- Из консоли Fluidd/Mainsail (нажать английскую букву `C` в fluidd)

!!! note
    *Значение указанное в скобках - это значение по умолчанию*
